# ais-tools
Tools for reading and writing AIS messages


## Install
```console
git clone https://github.com/SkyTruth/ais-tools
pip install ais-tools/
```

## Developing

```console
git clone https://github.com/SkyTruth/ais-tools
cd ais-tools
virtualenv venv
source venv/bin/activate
pip install -e .\[dev\]
pytest
```
